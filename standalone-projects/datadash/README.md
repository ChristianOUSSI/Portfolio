# DataDash

Dashboard Python Power AMC mock avec Recharts (Next.js/Vite compatible).

## Fonctionnalités (80% func)
- Charts interactifs (bar, pie)
- Filtres temps réel
- Métriques KPI
- Design glassmorphism
- Responsive

## Lancement
```bash
cd standalone-projects/datadash
npm install
npm run dev
```

## Vercel
npm run build → vercel.

Replace portfolio URL in App.tsx.
