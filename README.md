# Chris OUSSI Portfolio

Portfolio personnel construit avec Next.js 15 et Tailwind CSS.

## Mise en route

1. Installer les dépendances :
   ```bash
   npm install
   ```

2. Lancer le serveur de développement :
   ```bash
   npm run dev
   ```

3. Générer la version de production :
   ```bash
   npm run build
   ```

4. Lancer le serveur en production :
   ```bash
   npm run start
   ```

## Notes
- Remplacez les images de `public` par vos propres visuels.
- Utilisez `npm run lint:fix` pour corriger automatiquement les problèmes de lint.
- Thème clair/sombre : bouton dans la barre de navigation, préférence stockée en localStorage.
- Navigation par ancre de section et bouton retour en haut disponibles.
- Voir `VERCEL_DEPLOYMENT.md` pour déployer les projets standalone.
